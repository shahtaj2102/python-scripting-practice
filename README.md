## Featured Projects

### 1. 📱 Tip Calculator
Interactive tool to split bills with custom tip percentage.

**Skills Demonstrated:**
- User input (`input()`)
- Type conversion (`float()`, `int()`)
- F-string formatting
- Rounding (`round()`)

**Demo:**
```
Welcome to the tip calculator!
What was the total bill?
$100
What percentage tip would you like to give?
12
How many peopleto split the bill?
2
The amount to be paid by each person is
$56.0
```

[View Code](tip_calculator.py)


### 2. 🎢 Rollercoaster Tickets
Nested conditional pricing based on height/age/photo.

**Skills Demonstrated:**
- Nested if/elif chains
- Multiple conditions (`>=`, `<=`)
- String methods (`.upper()`)
- Dynamic billing logic
